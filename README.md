# Dashboard-LATAM
Creación de un dashboard simple para listar y agregar candidatos (Backend Express + Frontend React). 
Expresado en 2 enlaces diferentes: 
# Principal
<img width="1910" height="989" alt="imagen" src="https://github.com/user-attachments/assets/e1d89b2b-a290-41da-9ddd-f54e9e0fbf12" />
Visualización de portal de registro de candidato de postulación.
Los datos del registro están basados en datos habituales que se realizan en procesos de postulación comunes, en los cuales presentan información personal, así como también portales de presentacioń laboral y de registro de trabajo como LinkedIn y GitHub. Así como también un writebox para que el postulante pueda hacer ingreso de la motivación de su postulación. El Backend realizado con Express.js nos permite tener un registro minimalista del proceso de almacenamiento y de solicitud de datos de postulación, mientras que el Frontend en React para poder tener un UI interactivo y dinámico para la facilidad del usuario.
